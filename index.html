Git Command Cheat Sheet

Quick reference for common Git commands.

Basic Commands

  • git init - Initialize a new Git repository
  • git clone [url] - Clone a remote repository
  • git status - Show the working tree status
  • git add [file] - Add file contents to the index
  • git commit -m "[message]" - Record changes to the repository

Branching

  • git branch - List branches
  • git branch [name] - Create a new branch
  • git checkout [branch] - Switch to a branch
  • git merge [branch] - Merge a branch into the active branch
  • git branch -d [branch] - Delete a branch

History

  • git log - Show commit logs
  • git log --oneline - Compact commit logs
  • git diff - Show changes between commits
  • git show [commit] - Show various types of objects

Remote

  • git remote add [name] [url] - Add a remote repository
  • git push [remote] [branch] - Push to a remote repository
  • git pull [remote] [branch] - Fetch from and integrate with another repository
  • git fetch [remote] - Download objects from a remote repository
